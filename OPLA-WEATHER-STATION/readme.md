# 🌦️ Oplà IoT Weather Station

## 📌 Project Overview
An automated microclimate tracking system built using the Arduino Oplà IoT Kit. The system tracks local environmental metrics in real-time.

## 🛠️ Hardware Components
* **Carrier Board:** Arduino Oplà IoT Carrier
* **Microcontroller:** Arduino MKR WiFi 1010
* **Sensors:** On-board temperature, humidity, pressure.

## 📊 Tracked Metrics
* **Data Streams:** Temperature (°C), Relative Humidity (%), Atmospheric Pressure (hPa).

## 📁 Repository Structure
* `/Opla-Weather-Station.ino` - Main production firmware (C++).
* `/Schematics` - Circuit layout and pin mapping configurations.
